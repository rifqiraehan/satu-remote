Tentang SatuRemote

SatuRemote adalah sistem Remote Kontrol Universal berbasis ESP32 yang dirancang untuk menggantikan banyak remote elektronik di rumah Anda dengan satu perangkat terpusat yang praktis.

Satu Untuk Semua

Perangkat ini mempelajari serta memancarkan ulang kode infrared (IR) dari berbagai remote asli, memungkinkan kontrol TV, AC, dan proyektor melalui aplikasi smartphone via Wi-Fi.

Daftar Perangkat Elektronik

Televisi

Remote universal mempelajari dan memancarkan ulang kode IR standar TV, sehingga TV mengenali perintah sama seperti remote asli.

Air Conditioner

Merekam dan memancarkan ulang kode IR dari remote AC asli. AC merespons perintah tanpa perlu pengaturan tambahan pada unit AC.

Proyektor

Mengontrol fungsi dasar seperti power, input, dan menu pada mayoritas proyektor yang menggunakan sinyal infrared standar.

Langkah Umum Penggunaan

1

Nyalakan Remote

Pastikan perangkat SatuRemote telah mendapatkan daya (power).

2

Sambungkan WiFi

Hubungkan smartphone Anda ke Access Point (AP) yang dipancarkan oleh SatuRemote.

3

Buka Laman Web

Akses alamat saturemote.local di browser Anda.

4

Pilih Perangkat & Kontrol

Pilih device yang diinginkan, arahkan IR Blaster ke alat elektronik, dan tekan tombol virtual.

Koneksi WiFi & Akses Laman

1. Menghubungkan ke WiFi

Saat SatuRemote dinyalakan, ia akan memancarkan sinyal WiFi. Buka pengaturan WiFi di HP Anda dan cari jaringan bernama SatuRemote.

Default Password
saturemote

2. Mengakses Captive Portal

Setelah terhubung, biasanya halaman akan terbuka otomatis (Captive Portal). Jika tidak, buka browser (Chrome/Safari) dan ketik alamat berikut:

http://saturemote.local

Menambahkan Perangkat Baru

Opsi Template

Pilih dari daftar merek yang sudah tersedia di database SatuRemote. Berikan nama yang unik untuk memudahkan identifikasi (contoh: "TV Ruang Tamu").

  • Buka Menu Tambah Perangkat
  • Pilih Kategori (TV/AC/Proyektor)
  • Cari Merek
  • Simpan

Opsi Kustom

Jika merek perangkat Anda tidak terdaftar, pilih opsi "Kustom". Anda dapat menentukan tata letak tombol sendiri dan melakukan proses Learning.

Cara Learning Pada Remote

Fitur ini digunakan untuk "mengajarkan" SatuRemote kode IR dari remote asli Anda. Sangat berguna untuk perangkat dengan merek langka.

01
Siapkan Remote Asli

Pastikan remote asli berfungsi dan memiliki baterai.

02
Buka Mode Learning

Di aplikasi web, pilih tombol yang ingin diprogram, lalu aktifkan mode learning.

03
Tembakkan Sinyal

Arahkan remote asli ke Receiver SatuRemote dan tekan tombol.

Mengubah SSID & Password

Perhatian

Setelah mengubah pengaturan ini, SatuRemote akan melakukan Reboot. Anda harus menghubungkan ulang WiFi HP Anda ke nama jaringan (SSID) yang baru.

  1. Klik ikon garis tiga (Menu) di pojok kanan atas.
  2. Pilih opsi Pengaturan Jaringan.
  3. Masukkan SSID baru (Nama Jaringan).
  4. Masukkan Password baru (Minimal 8 Karakter).
  5. Tekan tombol Simpan dan Reboot.

Fitur Lainnya

Temukan Perangkat

Lupa menaruh alat? Aktifkan toggle ini di menu. Buzzer pada SatuRemote akan berbunyi nyaring untuk membantu Anda menemukannya.

Backup & Restore

Simpan konfigurasi remote Anda ke dalam file .json. Berguna untuk memindahkan data ke perangkat SatuRemote lain atau sebagai cadangan.
